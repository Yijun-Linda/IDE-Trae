## 一句话介绍

一个让心智障碍咖啡师能独立出品的小变化，部署在收银系统和云打印机之间，将"1x 拿铁（去冰，燕麦奶）"翻译为"拿冰杯→加冰块→加燕麦奶→倒浓缩"，帮助心智障碍员工独立完成咖啡制作。

## 核心问题与解决方案

### 问题：结果→过程的认知转换困难

在慈善咖啡店，心智障碍员工面临的挑战：
- 普通小票显示"1x 拿铁（去冰，燕麦奶）"，需要自己拆解成动作步骤
- 多订单同时涌入时，工作记忆超载，容易搞混、重复做
- 依赖辅导员口头指导，同步打断式支持成本高

### 解决方案：改造已有工具，而非增加新设备

**关键创新**：不是给员工增加新设备，而是改造他们已经在使用的工具（小票）。

**Before**：传统小票信息过载
```
美团外卖 #128
1x 拿铁（大杯，去冰，燕麦奶，半糖）
1x 冰美式（少冰）
1x 焦糖玛奇朵（大杯）
```

**After**：AI翻译后步骤清晰
```
  订单#128 第1/3杯
    冰拿铁
━━━━━━━━━━━━━━
□ 1. 拿冰杯大杯
□ 2. 不加冰
□ 3. 加燕麦奶
□ 4. 加半糖糖浆
□ 5. 萃取倒入
□ 6. 盖上出杯
━━━━━━━━━━━━━━
  5分钟后出下一杯
```

## 快速开始


### 启动服务

```bash
python src/webhook/handler.py
```

服务将在 http://localhost:5000 启动，提供以下接口：
- `POST /webhook/order` - 接收外卖平台订单
- `GET /queue/status` - 获取队列状态
- `POST /queue/skip` - 跳过当前杯

## 核心功能

### 1. AI订单拆解
- 支持10个SKU（5热饮+5冷饮）
- 识别规格：杯型、牛奶类型、甜度、冰量、加料
- 输出每步不超过10个汉字的动作指令

### 2. 极简小票模板
- 一次只显示一杯的制作步骤
- 带勾选框，完成后可打勾确认
- 删除所有非必要信息（价格、顾客信息、装饰线）

### 3. 时间延迟自动出单
- 每杯制作完成后等待5分钟
- 自动打印下一杯的指引小票
- 避免多订单同时涌入造成认知超载

### 4. 异常回退机制
- AI服务异常时自动切换回原始小票模式
- 确保系统可用性

## 设计原则

来自实地调研的核心洞察（杨老师，慈善咖啡店辅导员）：

1. **工作记忆是稀缺资源**：一次只处理一杯，避免认知超载
2. **被动接收优于主动检索**：小票自动打印，无需员工主动获取信息
3. **不要搞特殊**：不增加可见的新设备，改造已有工具而非插入新工具
4. **简短、流程化、短语式**：每步不超过10个字，使用动作动词
5. **优化的目的是没有优化**：辅助工具的使用频率应逐渐降低，而非成为永久依赖

## 待解决问题

### 问题 1：遗忘-主动

**现象**：如果员工根本不记得去看小票，所有提示失效

**杨老师的实践**：
- 墙上贴纸条：失败
- 拍视频给员工看：效果有限
- 视频检查机制：有效但需要辅导员在场

**可能的答案**：
- 轻度智力障碍者可能比中重度更容易建立看小票的习惯
- 旺季高负荷时，小票的存在本身就在"提醒他们看"
- 第三次忘记时给予更强的干预

**待验证**：这个假设是否成立？

### 问题 2：按需触发的设计

**现象**：当认知超载时，人很难自主判断"我需要帮助"

**杨老师的场景**：
> 中午她出去买个饭，刚走到半路，来自轻度心智障碍的员工的电话就打过来了

**这说明**：他们在慌乱时，目前会自主求助辅导老师

**可能的答案**：
- 被动推送解决"我忘了看"的问题
- 但无法解决"我根本不知道我需要看"的问题
- 需要外部触发（如队列延迟出单）

**待探索**：是否有更好的"按需触发"机制？

### 问题 3：用人单位动机

**现象**：普通咖啡店为什么要雇用轻度智力障碍员工？

**已识别的政策杠杆**：
- 残障就业保障金减免
- 增值税优惠
- CSR叙事价值（品牌形象）

**关键问题**：
- 轻度智力障碍在残障评定体系中属于几级？享受同等政策吗？
- 补贴额度是否足以覆盖"管理成本"溢价？
- 政策覆盖的岗位类型是什么？

**待探索**：政策细节 + 用人单位实际动机

## 核心洞察

> "社会化的本质是不要搞特殊。"
> 
> "优化的目的是没有优化。"
> 
> —— 杨老师，慈善咖啡店辅导员

AI认知翻译官不是在替代心智障碍者的工作，而是**翻译世界给他们能懂的语言**。

**长期目标**：不是让工具成为永久依赖，而是让他们在需要时看一眼/听一下，最终内化能力，不再需要工具。辅助的使用频率逐渐降低，而不是辅助本身的覆盖率提升。
