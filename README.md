# IDE-Trae

**IDE** = **I**ntellectual **D**isability **E**mployment\
**Trae** = 使用 [Trae](https://trae.ai) AI IDE 工具开发

***

## 项目背景

本项目是 **TRAE SOLO 挑战赛「Hello AI 科技致善」公益赛道** 参赛作品，聚焦**心智障碍群体就业支持**命题。

### 核心问题

心智障碍青年（包括唐氏综合征、自闭症等）在就业场景中面临巨大挑战。据实地观察，能在慈善商店正常就业的唐氏综合征或自闭症青年，可能不到该群体的 1%。

### 研究方向

探索如何利用 AI 技术（计算机视觉、语音交互、大语言模型）为心智障碍青年提供实时任务支持，降低就业门槛，扩大就业机会。
