# AI认知翻译官：视觉呈现设计方案

## 设计目标

- **3秒内理解项目价值**：直观展示"混乱→清晰"的对比
- **情感共鸣**：帮助心智障碍员工独立工作
- **网友投票友好**：适合社交媒体传播

***

### 视觉元素

**左侧（混乱）**：
- 背景：浅灰色，略显杂乱
- 小票：4-5张堆叠，文字密集
- 图标：❌ 或 😵（表示困惑）
- 色调：冷色调（灰、蓝）

**右侧（清晰）**：
- 背景：温暖米色
- 小票：单张，大号字体，留白充足
- 图标：✅ 或 😊（表示清晰）
- 色调：暖色调（咖啡棕、米黄）

**中间箭头**：
- 大箭头从左指向右
- 箭头上文字："AI翻译"

### 配色方案

| 元素 | 颜色 | 色值 |
|------|------|------|
| 主背景 | 温暖米色 | #F5F1E8 |
| 咖啡棕 | 主色调 | #6B4423 |

| 勾选绿 | 成功状态 | #4A7C59 |
| 混乱灰 | 对比色 | #9E9E9E |

### 字体建议

- **标题**：思源黑体 Bold / Montserrat Bold
- **正文**：思源黑体 Regular / Open Sans
- **小票文字**：等宽字体（模拟打印效果）

## 二、演示视频/动图（15-30秒）

### 分镜脚本

#### 场景一：多订单涌入（0-5秒）

**画面**：
- 打印机快速吐出4张小票
- 小票堆叠在桌上，杂乱无章
- 员工看着小票，表情困惑

**文字叠加**：
"当多个订单同时涌入..."

**音效**：
- 打印机快速打印声（急促）

#### 场景二：AI翻译（5-10秒）

**画面**：
- 动画：订单文字被AI拆解
- 一张张小票按顺序打印
- 每次只出一张

**文字叠加**：
"AI认知翻译官"
"把复杂订单拆解为简单步骤"

**动画效果**：
- 文字高亮："拿铁（去冰，燕麦奶）"
- 拆解动画：→ "拿冰杯" → "加冰块、加燕麦奶" → "萃取浓缩"

#### 场景三：员工独立完成（10-25秒）

**画面**：
- 员工看着小票，一步一步操作
- 特写：打勾动作 ✓
- 完成一杯，放入指定区域
- 5分钟后，下一张小票自动打印

**文字叠加**：
"一步一步，独立完成"
"5分钟后自动出下一单"

**音效**：
- 轻柔的背景音乐
- 打印机打印声（舒缓）

#### 结尾（25-30秒）

**画面**：
- 杨老师在远处微笑点头
- 员工自信地制作咖啡

**文字叠加**：
"社会化的本质是不要搞特殊"
"——杨老师，慈善咖啡店店长"

**Logo**：
AI认知翻译官 + 项目Slogan

### 视频规格

- **分辨率**：1080 x 1920px（竖屏，适合抖音/视频号）
- **帧率**：30fps
- **格式**：MP4 + GIF（动图版本用于微信/微博）

## 三、小票模板优化

### 打印效果优化

**字体选择**：
- 使用打印机支持的字体（宋体/黑体）
- 字号：标题16pt，正文14pt
- 加粗关键信息（步骤编号）

**排版优化**：
```
┌────────────────────┐
│ 订单#128 第1/4杯    │  ← 16pt 加粗
├────────────────────┤
│                    │
│ □ 1. 拿冰杯        │  ← 14pt
│                    │
│ □ 2. 加冰块、燕麦奶 │  ← 14pt
│                    │
│ □ 3. 萃取、倒入、盖盖│  ← 14pt
│                    │
├────────────────────┤
│ 5分钟后出下一杯     │  ← 12pt 灰色
└────────────────────┘
```

**勾选框设计**：
- 使用 Unicode 字符：□（未勾选）✓（已勾选）
- 或手绘风格勾选框（更友好）

**纸张选择**：
- 标准热敏纸（57mm或80mm宽度）
- 测试打印效果，确保文字清晰

## 四、项目展示页设计

### 页面结构（长图/网页）

**第一屏：问题场景**
- 大图：普通外卖小票堆叠
- 标题："当多个订单同时涌入..."
- 引用杨老师的话（增加真实感）

**第二屏：解决方案**
- 动图：AI翻译过程
- 标题："不是增加新设备，而是改造已有工具"
- 简洁架构图

**第三屏：效果对比**
- 左右对比图：Before vs After
- 数据：杨老师同步打断式支持减少50%

**第四屏：技术亮点**
- 图标：零新增设备、纯软件、低成本
- 一句话说明每个亮点

**第五屏：社会价值**
- 引用："社会化的本质是不要搞特殊"
- 愿景：让心智障碍者在真实职场中独立工作

### 长图规格

- **尺寸**：750 x 3000px（适合手机浏览）
- **格式**：JPG/PNG + PDF（打印版）
