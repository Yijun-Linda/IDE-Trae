## 1. 设计原则

### 1.1 认知友好原则

基于心智障碍员工的认知特点，小票设计遵循以下原则：

1. **一次一杯**：小票上只显示当前这一杯的信息，不显示总订单数
2. **步骤简短**：每步不超过10个汉字，使用动作动词
3. **视觉清晰**：大号字体、充足留白、高对比度
4. **可交互**：带勾选框，完成后可打勾确认

### 1.2 极简主义

删除所有非必要元素：
- ❌ 装饰线、花纹
- ❌ 店铺Logo（占用空间）
- ❌ 二维码、条形码
- ❌ 顾客信息（手机号、地址）
- ❌ 价格信息

保留核心元素：
- ✅ 订单号（用于核对）
- ✅ 当前杯号（第X杯）
- ✅ 饮品名称
- ✅ 制作步骤（带勾选框）
- ✅ 下一杯提示

## 2. 模板规格

### 2.1 纸张规格

| 规格 | 宽度 | 适用场景 | 每行字符数 |
|------|------|---------|-----------|
| 57mm | 57毫米 | 标准外卖小票 | 16个汉字 |
| 80mm | 80毫米 | 更清晰显示 | 24个汉字 |

### 2.2 字体规格

| 元素 | 字号 | 样式 | 说明 |
|------|------|------|------|
| 标题 | 16pt | 加粗 | 订单号+杯号 |
| 饮品名 | 14pt | 加粗 | 居中显示 |
| 步骤 | 14pt | 常规 | 左对齐 |
| 勾选框 | 14pt | Unicode | □ 字符 |
| 底部提示 | 12pt | 灰色 | 居中 |

### 2.3 间距规格

```
┌────────────────────┐
│                    │  ← 上边距 2mm
│ 订单#128 第1/4杯    │  ← 标题
│                    │  ← 空行 2mm
├────────────────────┤  ← 分隔线
│                    │  ← 空行 2mm
│ □ 1. 拿冰杯        │  ← 步骤
│                    │  ← 空行 3mm
│ □ 2. 加冰块、燕麦奶 │  ← 步骤
│                    │  ← 空行 3mm
│ □ 3. 萃取、倒入、盖盖│  ← 步骤
│                    │  ← 空行 2mm
├────────────────────┤  ← 分隔线
│ 5分钟后出下一杯     │  ← 底部提示
│                    │  ← 下边距 2mm
└────────────────────┘
```

## 3. 模板实现

### 3.1 数据模型

```python
from dataclasses import dataclass
from typing import List, Optional

@dataclass
class TicketData:
    """小票数据模型"""
    order_id: str                    # 订单号
    current: int                     # 当前杯数
    total: int                       # 总杯数
    drink_name: str                  # 饮品名称
    steps: List[str]                 # 制作步骤
    notes: Optional[str] = None      # 备注（规格信息）
    delay_minutes: int = 5           # 下一杯延迟时间

@dataclass
class TicketConfig:
    """小票配置"""
    width_mm: int = 57               # 纸张宽度
    title_font_size: int = 16        # 标题字号
    body_font_size: int = 14         # 正文字号
    small_font_size: int = 12        # 小字号
    line_spacing: int = 1            # 行间距倍数
    chars_per_line: int = 16         # 每行字符数
```

### 3.2 模板渲染器

```python
class TicketTemplate:
    """小票模板渲染器"""
    
    # ESC/POS 指令常量
    ESC = '\x1b'
    GS = '\x1d'
    ALIGN_CENTER = ESC + 'a' + '\x01'
    ALIGN_LEFT = ESC + 'a' + '\x00'
    BOLD_ON = ESC + 'E' + '\x01'
    BOLD_OFF = ESC + 'E' + '\x00'
    LINE_FEED = '\n'
    CUT_PAPER = GS + 'V' + '\x01'
    
    def __init__(self, config: Optional[TicketConfig] = None):
        self.config = config or TicketConfig()
        self.use_esc_pos = True  # 是否使用ESC/POS指令
    
    def render(self, data: TicketData) -> str:
        """
        渲染小票
        
        Args:
            data: 小票数据
            
        Returns:
            格式化后的小票内容（ESC/POS指令或纯文本）
        """
        if self.use_esc_pos:
            return self._render_esc_pos(data)
        else:
            return self._render_plain_text(data)
    
    def _render_esc_pos(self, data: TicketData) -> str:
        """使用ESC/POS指令渲染"""
        lines = []
        
        # 初始化
        lines.append(self.ESC + '@')  # 初始化打印机
        
        # 标题（加粗、居中）
        lines.append(self.ALIGN_CENTER)
        lines.append(self.BOLD_ON)
        lines.append(f"订单#{data.order_id} 第{data.current}/{data.total}杯")
        lines.append(self.BOLD_OFF)
        lines.append(self.LINE_FEED)
        
        # 饮品名
        lines.append(self.BOLD_ON)
        lines.append(data.drink_name)
        lines.append(self.BOLD_OFF)
        lines.append(self.LINE_FEED)
        
        # 分隔线
        lines.append(self.ALIGN_LEFT)
        lines.append("=" * self.config.chars_per_line)
        lines.append(self.LINE_FEED)
        
        # 步骤
        for i, step in enumerate(data.steps, 1):
            lines.append(f"□ {i}. {step}")
            lines.append(self.LINE_FEED)
            lines.append(self.LINE_FEED)  # 空行
        
        # 分隔线
        lines.append("-" * self.config.chars_per_line)
        lines.append(self.LINE_FEED)
        
        # 底部提示
        lines.append(self.ALIGN_CENTER)
        lines.append(f"{data.delay_minutes}分钟后出下一杯")
        lines.append(self.LINE_FEED)
        
        # 切纸
        lines.append(self.CUT_PAPER)
        
        return "".join(lines)
    
    def _render_plain_text(self, data: TicketData) -> str:
        """纯文本渲染（兼容性更好）"""
        lines = []
        width = self.config.chars_per_line
        
        # 标题居中
        title = f"订单#{data.order_id} 第{data.current}/{data.total}杯"
        lines.append(title.center(width))
        lines.append("")
        
        # 饮品名居中
        lines.append(data.drink_name.center(width))
        lines.append("")
        
        # 分隔线
        lines.append("=" * width)
        lines.append("")
        
        # 步骤
        for i, step in enumerate(data.steps, 1):
            lines.append(f"□ {i}. {step}")
            lines.append("")  # 空行
        
        # 分隔线
        lines.append("-" * width)
        lines.append("")
        
        # 底部提示
        footer = f"{data.delay_minutes}分钟后出下一杯"
        lines.append(footer.center(width))
        
        return "\n".join(lines)
```

### 3.3 模板工厂

```python
class TemplateFactory:
    """模板工厂"""
    
    @staticmethod
    def create_57mm_template() -> TicketTemplate:
        """创建57mm标准模板"""
        config = TicketConfig(
            width_mm=57,
            chars_per_line=16,
            title_font_size=16,
            body_font_size=14,
            small_font_size=12
        )
        return TicketTemplate(config)
    
    @staticmethod
    def create_80mm_template() -> TicketTemplate:
        """创建80mm标准模板"""
        config = TicketConfig(
            width_mm=80,
            chars_per_line=24,
            title_font_size=16,
            body_font_size=14,
            small_font_size=12
        )
        return TicketTemplate(config)
    
    @staticmethod
    def create_simple_template() -> TicketTemplate:
        """创建纯文本简化模板（最大兼容性）"""
        template = TicketTemplate()
        template.use_esc_pos = False
        return template
```

## 5. 测试验证

### 5.1 打印测试清单

```python
# tests/test_template.py

class TestTicketTemplate:
    """小票模板测试"""
    
    @pytest.fixture
    def sample_data(self):
        return TicketData(
            order_id="128",
            current=1,
            total=4,
            drink_name="冰拿铁",
            steps=["拿冰杯", "加冰块、加燕麦奶", "萃取倒入", "盖盖出杯"],
            notes="大杯",
            delay_minutes=5
        )
    
    def test_57mm_render(self, sample_data):
        """测试57mm模板渲染"""
        template = TemplateFactory.create_57mm_template()
        result = template.render(sample_data)
        
        assert "订单#128" in result
        assert "第1/4杯" in result
        assert "冰拿铁" in result
        assert "□ 1. 拿冰杯" in result
        assert "5分钟后出下一杯" in result
    
    def test_80mm_render(self, sample_data):
        """测试80mm模板渲染"""
        template = TemplateFactory.create_80mm_template()
        result = template.render(sample_data)
        
        lines = result.split('\n')
        # 80mm模板每行应该更长
        assert max(len(line) for line in lines) > 20
    
    def test_plain_text_render(self, sample_data):
        """测试纯文本渲染"""
        template = TemplateFactory.create_simple_template()
        result = template.render(sample_data)
        
        # 纯文本不应包含ESC/POS指令
        assert '\x1b' not in result
        assert '\x1d' not in result
    
    def test_step_limit(self):
        """测试步骤数量限制"""
        data = TicketData(
            order_id="999",
            current=1,
            total=1,
            drink_name="测试",
            steps=[f"步骤{i}" for i in range(20)],  # 20个步骤
            delay_minutes=5
        )
        
        template = TemplateFactory.create_57mm_template()
        result = template.render(data)
        
        # 应该能正常渲染，但实际打印可能需要分页
        assert "步骤1" in result
        assert "步骤20" in result
```

### 5.2 实地测试要求

| 测试项 | 通过标准 | 测试方法 |
|--------|---------|---------|
| 字体清晰度 | 1米外可清晰阅读 | 实际打印后距离测试 |
| 勾选框可见性 | □符号清晰可辨 | 目视检查 |
| 留白充足 | 不拥挤、易阅读 | 对比普通外卖小票 |

## 6. 使用示例

### 6.1 基本使用

```python
from src.printer.template import TicketTemplate, TicketData, TemplateFactory

# 创建模板
template = TemplateFactory.create_57mm_template()

# 准备数据
data = TicketData(
    order_id="128",
    current=1,
    total=4,
    drink_name="冰拿铁",
    steps=["拿冰杯", "加冰块、加燕麦奶", "萃取倒入", "盖盖出杯"],
    delay_minutes=5
)

# 渲染
content = template.render(data)
print(content)

# 发送到打印机
from src.printer.factory import PrinterFactory
printer = PrinterFactory.create("yilianyun", config)
result = printer.print_ticket(content)
```
